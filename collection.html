Process Of Data Collection And Storage

The Data for this project will be gotten from 4 major sources

Electricity monitor:

For now the project makes use of the OWL electricity monitoring device which makes use of sensors to monitor and record electricity energy consumption

Appliance Data:

information will be webscraped from validated web sources, by comparing the results from several sources to verify the accuracy of our results.

consumption data:

Considering that the electric meters in Nigeria are not smart meters, this information will be imputed manually by the users.

Solar Capacity data:

This Data will be collected from both the user device coordinates and name of location provided by the user. Here, the longitude and latitude of the user’s device can be derived by the device IP address of the device or the location data of the device. Also, the user can provide the name of location (address) to further ascertain the precise name of the location gotten from the long and Lat data from the user’s device. This coordinate will be collated in a tabular format. Plotting the coordinate of the user device in Open Street Map and further georeferencing these point data (Coordinate) in Google earth satellite imagery to determine where it falls and properly check the level of accuracy. Afterwards, building footprints of where the points fall will be digitized in a vector shapefile (polygon), where the area of the footprint will be calculated. Information on the available roof area of the user can be generated using the ArcGIS footprint calculator. Here, distance of the users to the solar providers can then be calculated in metres or kilometres. Also, travel time can also be determined using the distance from the location of the user and that of the solar provider. Vendor location data will be webscraped and actual geographical location can be extracted using google earth mapping service
